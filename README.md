# Q3- Minimax (Game AI)
This demonstrates the implementation of the Minimax algorithm to create an optimal Tic Tac Toe AI.

# Features
The AI assumes the opponent is always playing optimally and vice versa. This allows the AI to be unbeatable. 
The AI tries to maximise the score and the MIN opponent tries to minimise the score.
